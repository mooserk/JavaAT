Создать класс с единственной функцией (можно статической), которая принимает на вход ArrayList с числами от -9 до 9 включительно и возвращает значение типа String, в которой через запятую перечислены числа прописью через точку с запятой (например на входе список [-1, 8], функция вернёт "минус один;плюс восемь"). Если хоть одно из чисел в списке выходит за рамки разрешённого диапазона, функция должна вернуть "error"
2. Написать автотесты для проверки работы функции
3. Для сборки проекта использовать maven
4. Тестовый фреймворк использовать JUnit 5
5. Готовый проект выложить на github
6. Будет плюсом использование параметризированных тестов
